* 데이터 타입
 - scalar 또는 compound
 - 컴파일 시점에 모든 변수의 타입을 알아야 한다. (정적 타입 언어)

* Scalar Type
 - 정수 integer
 - 정수 리터럴 표현
 - 부동소수점 floating-point 타입
 - 숫자 기본 연산
 - 불린
 - char 타입 (유니코드 - 한글, 이모지)

* Compound Type

 - 튜플 (여러 타입을 넣을 수 있음)
 - 튜플 구조분해 (tuple destrusturing)
 - 튜플 아이템 접근 (인덱스로 접근)
 - 특별한 튜플 unit ( () ) void 같은 거임

 - 배열 array (동일한 타입을 여러 개)
 - 배열 길이가 고정이다. 길이가 달라지게 하려면 벡터(?)를 쓰면 된다.
